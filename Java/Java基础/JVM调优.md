jdk，jvm,jre区别
jdk java development kit 负责java编译
java 运行java文件
javac 编译java文件
jar 给java程序打包
javadoc 从java源码中提取注释并生成文档
jdb java调试器
jre java running environment，运行java虚拟机
jvm java virtual machine，识别class字节码文件并调用操作系统API，（跨平台的根本原因）
